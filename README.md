# bmi-calculator
A command-line interface tool to calculate a user's BMI based off their height and weight.
This tool was created for Assignment 2 of CSE 4283 at Mississippi State University.


## How to Use
Install `bmi.py` and `test.py` and place them in the same directory.
In your terminal, run `python -m bmi <weight> <height>`, replacing weight and height with your desired values.
To test, run `python -m pip install pytest` if it is not already installed, and `pytest test.py -v` to run the tests.
